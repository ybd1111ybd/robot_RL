- Grasp training strategy has been reframed as IK-gated RL rather than direct full-task PPO. IK is only a pre-training sanity gate for target geometry, action semantics, and reachable non-pathological postures; PPO should still learn from environment interaction, not follow an IK trajectory as the policy. `harness/tasks/grasp_ik_rl_plan.md` records the staged plan: IK sanity, Approach-only PPO, weak 6D orientation reward, gripper close/contact, then small randomization.

It supports multi-start IK selection with `--ik_random_starts` and `--min_joint_margin`, so target selection can reject solutions that reach position only by sitting on a joint limit. The script passed local syntax validation with `PYTHONPYCACHEPREFIX=/tmp/jz_pycache python -m py_compile jz_isaac_lab/scripts/tools/debug_grasp_ik.py`.
